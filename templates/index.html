Perguntas Frequentes

O Fatura Fácil armazena meus dados?

Não. O processamento é feito de forma temporária para gerar os gráficos. Seus dados de fatura não são salvos em nenhum banco de dados permanente.

Quais bancos são suportados?

Atualmente suportamos Nubank, Inter, Mercado Pago e Sicredi. Você pode sugerir novos bancos no botão "Sugerir Banco".

Quais os formatos de arquivo devo enviar?

Pode carregar a sua fatura em formato PDF (original do banco) ou CSV. Recomendamos o PDF para uma extração de dados mais precisa.

O serviço é gratuito?

Sim! O Fatura Fácil é uma ferramenta open-source e gratuita para ajudar na organização financeira pessoal.

A minha fatura não foi lida corretamente, o que fazer?

Certifique-se de que o ficheiro não está protegido por palavra-passe e que o banco é um dos suportados. Se o erro persistir, use o botão "Sugerir Banco" para nos enviar um exemplo (ocultando dados sensíveis).

A IA tem acesso aos meus dados bancários?

A IA processa apenas as descrições e valores das transações que o utilizador carrega para fornecer as análises. Não existe qualquer ligação direta com a sua conta bancária real.
